How to start learning Japanese

Here’s a beginner guide on how to learn Japan. Because there is no better time to start learning Japanese than right now. You may be in lockdown or you just spend a lot of time at home because everything is closed during this season or you want to stay at home because you want to be safe. This is my advice to beginners who want some kind of a guide to start learning Japanese. ^^

For the people who want to start learning this language, don’t worry about the big amounts that I’m talking about, which you have to learn. Learning a language requires dedication and time, a language is something very complicated, especially Japanese. Japanese is a very interesting language in my opinion and I really enjoyed the last 5 years of learning this amazing language. But it is quite an achievement to actually make a good/decent start at learning Japanese as there is sooo much information out there.

My advice is just, start learning and make progress every day, and don’t forget to review everything as learning a language is 10% learning and 90% reviewing/practicing. ;) Sadly there is no fast way to learn this language, bt please don’t give up! ^o^
